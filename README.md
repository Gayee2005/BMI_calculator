# BMI Calculator

## Overview
The BMI (Body Mass Index) Calculator is a simple application that calculates a person's BMI based on their height and weight. It helps users determine whether they are underweight, normal weight, overweight, or obese according to standard BMI ranges.

## Features
- User-friendly interface
- Calculates BMI using height and weight inputs
- Displays BMI value
- Shows BMI category:
  - Underweight
  - Normal Weight
  - Overweight
  - Obese
- Fast and accurate calculations

## Formula

BMI is calculated using the following formula:

BMI = Weight (kg) / (Height (m))²

## BMI Categories

| BMI Range | Category |
|------------|-----------|
| Below 18.5 | Underweight |
| 18.5 - 24.9 | Normal Weight |
| 25.0 - 29.9 | Overweight |
| 30.0 and above | Obese |
